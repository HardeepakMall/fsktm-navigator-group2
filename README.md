# FSKTM Navigator App

The FSKTM Navigator is a mobile application developed in Flutter designed to assist students and visitors in navigating Block A, Block B, and Block C of the Faculty of Computer Science and Information Technology (FSKTM) at UPM[cite: 1].

## 🚀 Getting Started

### Prerequisites
- **Flutter SDK:** Ensure you have the latest stable version of Flutter installed.
- **Android Studio / VS Code:** With the Flutter and Dart plugins enabled.
- **Android Emulator/Device:** To run the application.

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/HardeepakMall/fsktm-navigator-group2.git

# 2. Navigate to the project folder
cd fsktm_navigator

# 3. Install dependencies
flutter pub get

# 4. Run the application
flutter run
